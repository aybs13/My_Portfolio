## 🛠️ Getting Started

1. **Clone this repository**  
    Make sure **Node.js** and **Git** are installed globally.

2. **Installation**  
    Install the required dependencies:
    npm install

3. **Run the Application**
    Start the development server:
    npm start

## The app will run in development mode. Open http://localhost:3000 to view it in your browser. The page will automatically reload if you make edits.
